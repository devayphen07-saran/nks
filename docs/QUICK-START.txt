🎬 MOBILE AUTH FLOW:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1️⃣  LOGIN SCREEN (OTP)
      └─ User enters phone
      └─ sendOtp() → SMS sent
      └─ User enters OTP
      └─ verifyOtp() → Creates/logs in user

  2️⃣  PROFILE COMPLETION CHECK
      └─ Check: user.emailVerified ?
      └─ If NO → show ProfileCompletionScreen
      └─ If YES → skip to StoreSelection

  3️⃣  PROFILE COMPLETION SCREEN
      └─ Name (required)
      └─ Email + Password (optional)
      └─ profileComplete() → dispatched to Redux
      └─ Routes: emailVerification OR storeSelection

  4️⃣  STORE SELECTION SCREEN
      └─ useStores() → TanStack Query (auto-cached)
      └─ User selects store
      └─ storeSelect() → Redux updates activeStoreId
      └─ Navigate to StoreDashboard

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
